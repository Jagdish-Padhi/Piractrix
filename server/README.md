# Server Service

Express API service for Piractrix.

## Folder Intent

- `src/config`: environment and infra configuration
- `src/routes`: route definitions and grouping
- `src/controllers`: request handlers
- `src/services`: business logic
- `src/models`: database schemas
- `src/middlewares`: shared middleware
- `src/validators`: request payload validation schemas
- `src/jobs`: cron/scheduler jobs
- `src/utils`: generic helper utilities

## Commands

- `npm run dev`
- `npm run start`
- `npm run lint`
