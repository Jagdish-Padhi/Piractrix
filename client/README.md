# Client Service

React + Vite frontend for SportShield.

## Folder Intent

- `src/app`: app-level providers and wiring
- `src/pages`: route-level page components
- `src/components`: reusable UI and layout blocks
- `src/services`: API and external service clients
- `src/hooks`: reusable hooks
- `src/store`: global state containers
- `src/utils`: pure helper utilities
- `src/styles`: global and shared styles

## Commands

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`
